#Overview

->Built an interactive Tableau dashboard to analyze digital marketing campaign performance across platforms and campaign types using revenue, ad spend, conversions, CPA and ROAS metrics.
->The objective was to identify efficient campaigns and understand monthly performance trends.

#Key Responsibilities

->Cleaned and structured raw marketing dataset in Excel
->Designed KPI metrics (Revenue, ROAS, CPA, Conversions)
->Created platform-wise and campaign-wise performance comparisons
->Analyzed monthly revenue trends using time series visualization
->Derived business insights using annotations and summary observations

#Tools & Technologies

->Tableau
->Excel

#Technical Skills

->Data Cleaning
->Data Visualization 
->KPI Design 
->Dashboard Design 

#Analytical Skills

->Performance Comparison 
->Trend Analysis 
->Marketing ROI Analysis 
->Insight Generation 

#Key Highlights from Analysis

->Search campaigns showed higher efficiency compared to Display campaigns
->Certain platforms generated higher spend but lower returns
->Identified peak revenue month and performance fluctuations across the year
->Compared campaign types to understand cost effectiveness

#Summary

This project demonstrates the ability to transform raw marketing data into a clear visual dashboard and communicate insights for decision making.
